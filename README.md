# elixir-geo-2d-convex-hull

Compute a 2d convex hull of a set of points.

## Dependencies
* [elixir-geo-2d-types](https://github.com/ttvd/elixir-geo-2d-types)

## Usage
Add dependency in your mix.exs file:
```
def deps do
  [{:geo_2d_convex_hull, "~> 0.1"}]
end
```
